# 디자이너 포트폴리오 웹사이트

이 폴더에는 디자이너 포트폴리오용 정적 웹사이트가 포함되어 있습니다.

## 포함된 파일

- `index.html` - 메인 페이지
- `styles.css` - 사이트 스타일
- `script.js` - 간단한 연락처 폼 동작

## 실행 방법

1. `index.html` 파일을 브라우저에서 열거나
2. 로컬 웹 서버를 사용하여 실행합니다.

예:

- VS Code에서 Live Server 확장 사용
- 혹은 `python -m http.server` 명령어로 실행
